# 太极场生命能量模型 — 数理推导
# Taiji Field Life Energy Model — Mathematical Derivation

## 一、理论基石

本模型基于易宇本源体系，将传统太极阴阳理论转化为可计算的数理框架。

**核心假设：**
- 生命能量状态可由 N 个可观测维度向量描述
- 每个维度具有阴阳属性，对应相空间中的两个互补子空间
- 生命系统的稳态对应阴阳能量的动态平衡

---

## 二、维度定义

设生命能量维度集合为 D = {d₁, d₂, ..., d₈}

| 维度 | 阴阳属性 | 权重 wᵢ | 说明 |
|------|----------|----------|------|
| 睡眠质量 | 阴 | 1.2 | 阴精复原，权重最高 |
| 运动活力 | 阳 | 1.0 | 阳气升发 |
| 情绪稳定 | 阴 | 1.3 | 心神安定，权重最高 |
| 饮食均衡 | 阴 | 1.0 | 后天之本 |
| 社交连接 | 阳 | 0.9 | 阳气外发 |
| 精神修养 | 阴 | 1.1 | 精神内守 |
| 创造表达 | 阳 | 1.0 | 阳气舒发 |
| 自然接触 | 阳 | 0.8 | 天地之气 |

权重依据：阴属性维度权重总和 = 4.6，阳属性维度权重总和 = 3.7，符合「阴主静、阳主动」的生理节律特征。

## 三、能量计算公式

### 3.1 阴阳分组均值

**阳性能量均值：**
> ȳ_yang = Σ(wᵢ × vᵢ) / Σwᵢ，其中 dᵢ ∈ 阳

**阴性能量均值：**
> ȳ_yin = Σ(wᵢ × vᵢ) / Σwᵢ，其中 dᵢ ∈ 阴

其中 vᵢ ∈ [0, 100] 为用户滑块输入值。

### 3.2 平衡度系数

> B = 1 - |ȳ_yang - ȳ_yin| / 100

B ∈ [0, 1]，B = 1 表示阴阳完全平衡，B = 0 表示阴阳极端失衡。

### 3.3 综合能量得分

> E = round( V_total × (0.6 + 0.4 × B) )

其中 V_total = (ȳ_yang × Σw_yang + ȳ_yin × Σw_yin) / (Σw_yang + Σw_yin)

**系数解释：**
- 0.6：基础能量系数，代表各维度绝对值的贡献
- 0.4：平衡奖励系数，奖励阴阳协调状态
- 该系数结构保证：总分相同的情况下，阴阳越平衡，最终得分越高

## 五、模型局限与展望

1. **权重固化问题**：当前权重为专家经验设定，缺乏大规模数据标定
2. **线性假设**：各维度独立加权，未考虑维度间的非线性耦合
3. **数据验证**：需要与生理指标（心率变异性、皮质醇水平等）做相关性验证

**后续方向：**
- 引入机器学习标定权重
- 对接可穿戴设备实时数据
- 建立个人能量基线（baseline）而非绝对分值

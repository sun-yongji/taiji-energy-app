# 贡献指南 · 太极场生命能量模型

感谢你对本项目的关注！本文档说明如何参与前端应用的开发与改进。

## 快速开始

1. Fork 本仓库
2. 克隆到本地：`git clone https://github.com/sun-yongji/taiji-energy-app.git`
3. 直接用浏览器打开 `index.html` 即可预览
4. 开始开发

## 贡献类型

### 前端开发
- UI/UX 交互优化
- Canvas 动画效果增强
- 响应式布局适配
- 移动端优化

### 功能扩展
- 新增生命维度评估
- 数据持久化（LocalStorage / 后端 API）
- 生理指标接入接口
- 多语言支持

### 文档
- 使用教程与示例
- 中英文翻译
- API 文档（如有后端对接）

## 开发流程

1. Fork → 创建分支：`git checkout -b feat/your-feature`
2. 编写代码
3. 在浏览器中测试
4. 提交：`git commit -m "feat: add xxx"`
5. 推送 → 提交 Pull Request

## Commit 规范

```
<type>: <description>
```

| type | 用途 |
|------|------|
| feat | 新功能 |
| fix | 修复 bug |
| ui | UI/UX 改进 |
| docs | 文档 |
| refactor | 重构 |

## 代码规范

- 使用语义化 HTML
- CSS 命名使用 BEM 规范
- JavaScript 使用 ES6+ 语法
- 保持代码注释清晰

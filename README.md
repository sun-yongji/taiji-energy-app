## 项目简介

本模型将东方太极阴阳理论转化为可量化的能量评估工具。通过八个核心生命维度（睡眠、运动、情绪、饮食、社交、精神、创造、自然接触）的实时输入，计算个体的阴阳能量平衡态与综合生命能量得分。

This model translates Eastern Taiji yin-yang theory into a quantifiable energy assessment tool. Through real-time input across eight core life dimensions, it calculates an individual's yin-yang energy balance and comprehensive life energy score.

## 核心特性

- **八维能量评估** — 阴阳分类、加权计算、实时反馈
- **太极动画可视化** — Canvas 实时渲染阴阳鱼动态
- **能量雷达图** — 八维能力全景展示
- **七日能量时间线** — 能量趋势追踪
- **个性化调养建议** — 基于短板维度的中医养生建议
